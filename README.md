## slackscope

Read-first Slack CLI for listing chats/DMs and reading message history and threads.

## Install

```bash
uv tool install git+https://github.com/SZRabinowitz/slackscope
```

## Install skill (skills.sh)

Install the skill package from this repo:

```bash
npx skills add SZRabinowitz/slackscope
```

## Quick start

```bash
slack auth login <workspace>
slack chat list --limit 10
```
